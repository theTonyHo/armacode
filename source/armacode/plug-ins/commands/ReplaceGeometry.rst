ReplaceGeometry
---------------
Replace the object with the geometry from another object.
This is useful when the GUID of the object is critically important and needs to be retained.
Input is deleted on success.

Allowed object type:

- Curve
- Extrusion
- Brep (Surface/Polysurface)
- Mesh
