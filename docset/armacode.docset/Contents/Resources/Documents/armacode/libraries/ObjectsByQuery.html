ObjectsByQuery¶

ObjectsByQuery(name=None, layerName=None, layout=None, filter=0, include_grips=False, include_lights=False, is_instances=False, include_reference=False, include_selected=False, state=0, **kwargs)¶

Select objects by various properties and attributes. Supply a list of object ids in parameter OBJECT_IDS to search specifically within that set of objects. Return all objects if no parameters specified.
