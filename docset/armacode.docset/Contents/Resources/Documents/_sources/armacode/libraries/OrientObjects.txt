OrientObjects
-------------

.. py:Function:: OrientObjects(object_ids, referencePlane, targetPlane, copy=False)


Orients a single object based on input points

:param object_ids: List of Guid. The identifiers of an object
:param referencePlane: list of 3-D reference points
:param targetPlane: list of 3-D target points
:param copy[opt]: True = copy object

:returns: Transformed Object Ids on success.
