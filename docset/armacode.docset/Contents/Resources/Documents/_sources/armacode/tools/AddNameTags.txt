AddNameTags
-----------
Add a Text Dot object as a name tag for a object.

Position of the name tag is a combination of the following options:

* Bounding Box
* Curve Mid Point
* Curve Vertices
* Surface Center Parameter (0.5,0.5)
* Area Centroid (Surface, Curve, Mesh)
* Brep Vertices
* Brep Closest Point
