BatchConvertFilesTo3DPDF
------------------------
Batch Convert compatible files into 3D PDF.
3D PDF will have a default Perspective View.
Optionally, user can prepare Perspective View as desired.
Target PDF file name matchs with the input file.

Dependencies:
    SimLab 3D PDF Plugin
