AnalysisSurfaceArea
-------------------
Analyse Area of Surfaces, and Open Polysurfaces

Add an Attribute UserText to the object.
Add a Text Dot representing the value at the centroid
Generates a Set of layers representing different bands of values.
Duplicate and place each object on the layer corresponding to their calculated value for selection.
